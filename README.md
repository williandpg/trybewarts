<h1><strong>Trybewarts</strong></h1>

<h2><strong>Descrição</strong></h2>
<p>
  O objetivo deste projeto é criar uma página de formulário interativa para a Escola de Magia de Trybewarts, onde estudantes podem enviar seus feedbacks.  
  A página foi construída utilizando HTML, CSS e JavaScript, unindo conceitos de formulário, validação de campos e interatividade com uma temática inspirada na obra "Harry Potter" de J. K. Rowling.
</p>
<p>
  Mesmo sem a necessidade de conhecimento prévio sobre o universo original, o projeto propõe uma experiência mágica, temática e envolvente para os usuários, reforçando conceitos fundamentais de desenvolvimento front-end.
</p>

<h2><strong>Funcionalidades</strong></h2>
<ul>
  <li>
    <strong>Formulário de Feedback.</strong>  
    Os estudantes podem preencher um formulário com campos como nome, email, casa de bruxo(a), ano de estudo e mensagem de feedback.
  </li>
  <li>
    <strong>Validação de Campos.</strong>  
    Antes do envio, os campos são validados para garantir preenchimento obrigatório, formato correto de email e demais critérios definidos no projeto.
  </li>
  <li>
    <strong>Mensagens de Erro.</strong>  
    Em caso de preenchimento incorreto ou incompleto, mensagens de erro claras orientam o usuário sobre o que deve ser corrigido.
  </li>
  <li>
    <strong>Experiência Temática.</strong>  
    A página é personalizada com elementos visuais e textuais inspirados na Escola de Magia de Trybewarts, proporcionando uma interface imersiva e divertida.
  </li>
  <li>
    <strong>Interatividade com JavaScript.</strong>  
    Uso de eventos, manipulação do DOM e feedback dinâmico para melhorar a experiência do usuário durante o preenchimento do formulário.
  </li>
</ul>

  <h1><strong>Trybewarts</strong></h1>

  <h2><strong>Description</strong></h2>
  <p> 
    The goal of this project is to build an interactive feedback form page for the Trybewarts School of Magic, where students can submit their opinions and comments.  
    The page was created using HTML, CSS and JavaScript, combining front-end fundamentals with a magical, themed experience inspired by the "Harry Potter" universe.
  </p>

  <h2><strong>Features</strong></h2>
  <ul>
    <li>
      <strong>Feedback Form.</strong>  
      Interactive form where students can fill in name, email, wizard house, school year and feedback message.
    </li>
    <li>
      <strong>Field Validation.</strong>  
      Required fields and email format are validated before submission to ensure proper data entry.
    </li>
    <li>
      <strong>Error Messages.</strong>  
      Clear error messages guide users whenever some information is missing or invalid.
    </li>
    <li>
      <strong>Themed Experience.</strong>  
      Visual elements and text inspired by the Trybewarts School of Magic, creating an immersive and fun interface.
    </li>
    <li>
      <strong>JavaScript Interactivity.</strong>  
      Use of events, DOM manipulation and dynamic behavior to enhance user experience while filling the form.
    </li>
  </ul>
